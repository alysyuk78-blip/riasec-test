# RIASEC — тест профорієнтації (O*NET Interest Profiler)

> Цей файл Claude Code читає автоматично на старті. Тут — увесь контекст проєкту,
> щоб продовжити роботу без втрати рішень, ухвалених у попередніх сесіях.

## Що це
Односторінковий інтерактивний тест профорієнтації українською на базі офіційного
**O*NET Interest Profiler Short Form**. Один самодостатній файл `riasec-test.html`
(HTML + CSS + чистий JS, без збірки, без залежностей, без бекенду).

## Наукова основа — НЕ змінювати без вагомої причини
- 60 пунктів = **O*NET Interest Profiler Short Form** (Rounds, Su, Lewis & Rivkin, 2010),
  Національний центр розвитку O*NET / Міністерство праці США. Суспільне надбання (CC BY 4.0) —
  використання й переклад дозволені.
- Модель **RIASEC Голланда (1997)**: 6 типів — R, I, A, S, E, C.
- Пункти перекладені **дослівно**, у форматі «конкретна робоча дія» (це відрізняє O*NET від
  абстрактних опитувальників). Комерційні інструменти Голланда (SDS, VPI) — під копірайтом,
  їх відтворювати НЕ можна; O*NET IP — можна.
- **Порядок пунктів** — автентичне чергування онлайн-версії: пари, що циклічно проходять
  R-I-A-S-E-C (5 раундів × 12 = 60). Зменшує halo-ефект. НЕ сортувати за типами.
- **Шкала відповідей** — «подобається / не подобається», 5 балів із серединою «Не знаю»
  (оригінал: Strongly Dislike … Strongly Like, Unsure).

## Логіка підрахунку
- Кожен пункт = 1..5. Сума по типу (10 пунктів) → нормалізація:
  `pct = (сума − n) / (n · 4) · 100` → діапазон 0..100 % (n = к-сть пунктів типу).
- **Код Голланда** = топ-3 типи за балом. Рівні бали → стабільний порядок RIASEC.
- **«Надійність результату»** = функція розкиду балів (spread = max − min):
  <15 низька · 15–32 середня · ≥32 висока. Плаский профіль чесно позначається як орієнтовний.

## Дизайн
- Мінімалізм у стилі Apple: системний шрифт SF (`-apple-system, BlinkMacSystemFont, "SF Pro…"`,
  fallback Inter), фон `#fff`/`#f5f5f7`, текст `#1d1d1f`, синій акцент `#0071e3`.
- Кольори RIASEC — лише там, де кодують дані (гексагон, шкали інтересів).
- Емодзі-шкала — **Twemoji** (2D SVG) з CDN jsDelivr, версію зафіксовано `@15.1.0`.
  Якщо CDN недоступний → авто-fallback на знаки ○ ◔ ◑ ◕ ● (функція `emojiFail`). Завжди читабельно.

## Обмеження середовища
- Артефакт claude.ai: **НЕ використовувати** localStorage/sessionStorage. Стан лише в пам'яті JS.
- Гексагон — адаптивний SVG (`width:100%;max-width:300px`), не ламається на телефонах.
- Є `@media print`: зберігає кольори смуг, ховає навігацію (для «Зберегти → PDF»).

## Перевірка — тримати зеленою після кожної правки
- `node --check` на вмісті `<script>` — без синтаксичних помилок.
- Симуляція підрахунку: усі «ні» → 0 %; усі «так» → 100 %; 20 000 випадкових проходжень —
  жодного значення поза [0,100], без NaN.
- Рівно 10 пунктів × 6 типів = 60, без дублікатів; кожен блок = по 2 пункти кожного типу.

## Як запускати / тестувати
- Відкрити `riasec-test.html` у браузері.
- Швидка перевірка JS: витягнути вміст `<script>` і прогнати `node --check`.
